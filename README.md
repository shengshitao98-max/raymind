# RayMind 智能农田机器人

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![ROS2](https://img.shields.io/badge/ROS2-Humble-green)
![Platform](https://img.shields.io/badge/Platform-Jetson%20Orin-orange)

基于ROS2的智能农田机器人控制系统，支持杂草检测、自主导航、远程控制。

```
├── raymind_robot/
```

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/shengshitao98-max/raymind.git
cd raymind

# 运行安装脚本
chmod +x install.sh
./install.sh
```

### 启动

```bash
# 机器人端
source /opt/ros/humble/setup.bash
ros2 launch raymind_robot robot.launch.py

# 控制端
python3 raymind/gui.py
```

## ⚠️ 风险评估

| 风险 | 应对措施 |
|------|----------|
| SLAM在极端农田失效 | GNSS-RTK绝对位置约束 |
| 杂草检测泛化能力不足 | 预留在线学习接口 |
| 激光打击精度受振动影响 | 主动减震+运动补偿 |
| 系统实时性 | TensorRT加速+资源分配 |
