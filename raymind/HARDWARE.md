# RayMind 智能农田作业机器人 - 硬件方案

## 📋 系统概述

RayMind 是一套融合环境深度感知、自主导航、高精度杂草识别与定位、以及激光精准打击四大核心技术的智能农田作业机器人系统。

## 🔌 GPIO引脚分配

```
Jetson Orin Nano 40-pin Header:

I2C (总线1):
├── SDA → ICM-20948 (IMU)
└── SCL → BNO085 (备选IMU)

GPIO:
├── GPIO33 (PWM) → L298N ENA (左轮使能)
├── GPIO35 → L298N IN1 (左轮正转)
├── GPIO37 → L298N IN2 (左轮反转)
├── GPIO36 (PWM) → L298N ENB (右轮使能)
├── GPIO38 → L298N IN3 (右轮正转)
├── GPIO40 → L298N IN4 (右轮反转)
├── GPIO31 (PWM) → 激光器触发
├── GPIO32 (PWM) → 激光器功率
├── GPIO29 (PWM) → 舵机1 (航向)
└── GPIO31 (PWM) → 舵机2 (俯仰)

USB:
├── USB0 → USB工业相机
├── USB1 → RS-LiDAR-16
└── USB2 → NEO-6M GPS
```

## ⚠️ 安全警告

1. **激光安全**: 10W激光对眼睛有害，操作时必须佩戴防护眼镜
2. **电气安全**: 48V电池操作需专业人员，防止短路
3. **机械安全**: 测试时保持安全距离
4. **环境安全**: 农田作业注意防水防尘
